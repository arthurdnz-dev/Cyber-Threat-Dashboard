Cyber Defense System

Cyber Defense System é um dashboard interativo de monitoramento de ameaças cibernéticas em tempo real. O sistema simula eventos de segurança, analisa níveis de risco e apresenta os dados em uma interface visual inspirada em centros de operações de segurança.

O projeto foi desenvolvido com foco em experiência do usuário, visualização de dados e simulação de cenários reais de cibersegurança.

Funcionalidades

Monitoramento contínuo de eventos simulados de ataque
Classificação de risco em níveis baixo médio e alto
Visualização de ataques em mapa global com geolocalização simulada
Gráfico dinâmico de atividade de ataques em tempo real
Sistema de logs com atualização contínua
Filtro de eventos por nível de risco
Simulação de ataque massivo sob demanda
Alerta visual para ameaças críticas com controle de frequência
Interface com efeitos visuais inspirados em sistemas de segurança profissionais

Tecnologias utilizadas

JavaScript
HTML
CSS
Chart.js para visualização de dados
Leaflet para geolocalização e mapa interativo

Objetivo do projeto

O objetivo deste projeto é demonstrar a capacidade de construir interfaces interativas, manipular eventos em tempo real e representar dados de forma visual e intuitiva, simulando o comportamento de ferramentas utilizadas em ambientes reais de segurança da informação.

Como executar

Faça o download ou clone o repositório
Abra o arquivo index.html no navegador
O sistema iniciará automaticamente a simulação de eventos

Possíveis evoluções

Integração com APIs reais de threat intelligence
Sistema de autenticação e controle de acesso
Persistência de dados com banco de dados
Dashboard com métricas avançadas e análise histórica
Integração com backend para processamento real de eventos
